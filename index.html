How to Use

📌 Create Loan: Fill form with borrower details and click Create

📋 View Loans: All loans appear on Home tab

👁️ View Details: Tap any loan to see full details

✏️ Edit: Only active loans can be edited

✅ Close: Mark loan as closed when complete

🗑️ Delete: Only closed loans can be deleted

💾 Data: All saved locally on your phone
