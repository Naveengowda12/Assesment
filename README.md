# 💰 Loan Manager

A simple, user-friendly web application to track loans and manage payments.

## ✨ Features

- 📋 Create and manage loans
- 💰 Track payments month by month
- 📊 View loan summary statistics
- 💾 Data saved locally in browser
- 📱 Fully responsive mobile design

## 🚀 Live Demo

[View Loan Manager](https://naveengowda12.github.io/Assesment/loan-manager.html)

## 📖 How to Use

1. **Create a Loan**: Go to the "New" tab and fill in borrower details
2. **View Loans**: All active loans appear on the Home tab
3. **View Details**: Click any loan to see full details and payment history
4. **Add Payments**: Record collected payments for each loan
5. **Close Loan**: Mark a loan as closed when complete
6. **Delete**: Remove closed loans from the list

## 💾 Data Storage

All data is stored locally in your browser using localStorage. No server or account needed!

## 🛠️ Technology

- HTML5
- CSS3 (with gradient and animations)
- Vanilla JavaScript
- LocalStorage API
